Ubicacion Almacenaje::hueco_mas_adecuado ( const Contenedor &  c)

Encuentra el hueco más adecuado según la estrategia BEST_FIT para insertar un Contenedor.

Precondición
La lista de huecos h no está vacía y 1<=l<=3
Postcondición
El resultado es la ubicación <-1,0,0> si no hay un hueco disponible en el área de almacenaje y, por lo tanto, se colocará el Contenedor en el área de espera; en caso contrario el resultado es la ubicación dónde empieza el hueco con menor longitud, número menor de hilera y número menor de plaza posible en la que es posible insertar el contenedor
