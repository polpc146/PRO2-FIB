Consultora de los contenedores que hay en el área de espera esperando a ser insertados.

Precondición
b se pasa como Cierto si anteriormente se ha podido insertar un contenedor del área de espera o si es el primera vez que se consulta, se pasa como Falso si no se ha podido ubicar un contenedor anterior, c es el primer contenedor de la lista si es la primera llamada a la funcion o es el ultimo contenedor de la lista que se ha consultado
Postcondición
c es el siguiente contenedor que tocará observar si cabe en el área de almacenaje o es un contenedor vacío si ya no hay más contenedores que analizar
