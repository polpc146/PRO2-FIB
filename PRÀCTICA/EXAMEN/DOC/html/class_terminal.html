void Terminal::insertar_contenedor ( const Contenedor &  c)

Añade un Contenedor a la Terminal.

Precondición
No existe el Contenedor c en la Terminal
Postcondición
Se inserta el Contenedor en el AreaDeEspera si no tiene hueco, en caso contrario se coloca en el AreaDeAlmacenaje en la Ubicacion más adecuada según la estrategia BEST_FIT, se actualizan AreaDeEspera, ContenedoresAlmacenaje y Huecos y se recorre el AreaDeEspera por si se puede colocar algún Contenedor más. En ambos casos se imprime por pantalla la Ubicacion donde se ha colocado
