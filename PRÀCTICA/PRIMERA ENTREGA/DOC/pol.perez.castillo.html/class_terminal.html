void Terminal::insertar_AreaDeEspera ( const list< Contenedor > &  vespera)

Añade un Contenedor del area de espera al area de almacenaje.

Precondición
La lista AreaDeEspera no está vacía
Postcondición
Se han inserido todos los Contenedores (empezando por el último que entró al area de espera) al AreaDeAlmacenaje si tienen un hueco disponible y se actualiza la lista de Huecos si se ha conseguido insertar algún Contenedor

◆ insertar_contenedor()

void Terminal::insertar_contenedor ( const Contenedor &  c)

Añade un Contenedor a la Terminal.

Precondición
No existe el Contenedor c en la Terminal
Postcondición
Se inserta el Contenedor en el AreaDeEspera si no tiene hueco, en caso contrario se coloca en el AreaDeAlmacenaje en la Ubicacion más adecuada según la estrategia BEST_FIT, se actualizan las listas AreaDeEspera, ContenedoresAlmacenaje y Huecos y se recorre el AreaDeEspera por si se puede colocar algún Contenedor más. En ambos casos se imprime por pantalla la Ubicacion donde se ha colocado

Ubicacion Terminal::hueco_mas_adecuado ( const list< Segmento > &  h,
int  l 
) const

Encuentra el hueco más adecuado según la estrategia BEST_FIT para insertar un Contenedor.

Precondición
La lista de huecos h no está vacía y 1<=l<=3
Postcondición
El resultado es la Ubicacion <-1,0,0> si no hay un hueco disponible en el area de almacenaje y, por lo tanto, se colocará el Contenedor en el area de espera; en caso contrario el resultado es la Ubicacion dónde empieza el hueco con menor longitud, número menor de hilera y número menor de plaza posible en la que es posible insertar el contenedor
